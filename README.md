# 📄 Resume Analyzer

An ATS-based web application that analyzes resumes, ranks candidates, and identifies missing skills.

## 🚀 Live Demo
https://resume-analyzer-aditi.streamlit.app

## 💡 Features
- Resume ranking
- Skill matching
- Missing skills detection
- Suggested skills improvement

## 🛠 Tech Stack
- Python
- Streamlit
- PyPDF2
