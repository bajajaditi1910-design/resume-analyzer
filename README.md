# 📄 Rule-Based ATS Resume Analyzer (SWE Roles)

A web-based Resume Analyzer that simulates an Applicant Tracking System (ATS) using rule-based logic.  
It analyzes resumes, ranks candidates, and identifies missing skills based on job descriptions for Software Engineering roles.

## 🚀 Live Demo
https://resume-analyzer-aditi.streamlit.app

## 💡 Features
- Resume ranking based on match score
- Skill extraction using rule-based matching
- Missing skills detection
- Suggested skills for improvement
- Clean and interactive UI
- Multi-resume comparison

## ⚙️ How It Works
- Extracts text from PDF resumes
- Matches predefined technical skills
- Compares with job description
- Calculates match percentage
- Highlights missing skills

## 🛠 Tech Stack
- Python
- Streamlit
- PyPDF2
- Regex (Rule-based matching)

## 🎯 Use Case
Designed for analyzing resumes for **Software Engineering roles**, enabling recruiters to efficiently shortlist candidates through ranking, while helping applicants identify skill gaps and improve their profiles based on job requirements.
